# Excel to Matrix 

take a .xlsx file and turn it into a matrix using C++

OpenXLSX depends on:  
* PugiXML
* Zippy (C++ wrapper around miniz)

## how to use
put data in data directory      
cd to build/output  
run cmake --build ..    
run ./excel-to-matrix   

enter file name with extension when prompted.   
enter rows  
enter cols
